<span class="s1">&lt;p&gt;</span><span class="s0">Setting the display mode in </span><span class="s1">&lt;em&gt;</span><span class="s0">pygame</span><span class="s1">&lt;/em&gt; </span><span class="s0">creates a visible image surface</span>

<span class="s0">There are special functions needed in the </span><span class="s1">&lt;a </span><span class="s2">class</span><span class="s3">=&quot;tooltip reference internal&quot; </span><span class="s2">href</span><span class="s3">=&quot;../ref/display.html#module-pygame.display&quot; </span><span class="s2">title</span><span class="s3">=&quot;&quot;</span><span class="s1">&gt;&lt;code </span><span class="s2">class</span><span class="s3">=&quot;xref py py-mod docutils literal notranslate&quot;</span><span class="s1">&gt;&lt;span </span><span class="s2">class</span><span class="s3">=&quot;pre&quot;</span><span class="s1">&gt;</span><span class="s0">pygame.display</span><span class="s1">&lt;/span&gt;&lt;/code&gt;&lt;span </span><span class="s2">class</span><span class="s3">=&quot;tooltip-content&quot;</span><span class="s1">&gt;</span><span class="s0">pygame module to control the display window and screen</span><span class="s1">&lt;/span&gt;&lt;/a&gt;</span>
<span class="s0">module to keep the image surface contents updated on the monitor.</span><span class="s1">&lt;/p&gt;</span>
<span class="s1">&lt;p&gt;</span><span class="s0">Setting the display mode in </span><span class="s1">&lt;em&gt;</span><span class="s0">pygame</span><span class="s1">&lt;/em&gt; </span><span class="s0">is an easier task than with most</span>
<span class="s0">graphic libraries.</span>
<span class="s0">The advantage is if your display mode is not available,</span>
<span class="s1">&lt;em&gt;</span><span class="s0">pygame</span><span class="s1">&lt;/em&gt; </span><span class="s0">will emulate the display mode that you asked for.</span>
<span class="s1">&lt;em&gt;</span><span class="s0">Pygame</span><span class="s1">&lt;/em&gt; </span><span class="s0">will select a display resolution and color depth that best matches</span>
<span class="s0">the settings you have requested,</span>
<span class="s0">then allow you to access the display with the format you have requested.</span>
<span class="s0">In reality, since the </span><span class="s1">&lt;a </span><span class="s2">class</span><span class="s3">=&quot;tooltip reference internal&quot; </span><span class="s2">href</span><span class="s3">=&quot;../ref/display.html#module-pygame.display&quot; </span><span class="s2">title</span><span class="s3">=&quot;&quot;</span><span class="s1">&gt;&lt;code </span><span class="s2">class</span><span class="s3">=&quot;xref py py-mod docutils literal notranslate&quot;</span><span class="s1">&gt;&lt;span </span><span class="s2">class</span><span class="s3">=&quot;pre&quot;</span><span class="s1">&gt;</span><span class="s0">pygame.display</span><span class="s1">&lt;/span&gt;&lt;/code&gt;&lt;span </span><span class="s2">class</span><span class="s3">=&quot;tooltip-content&quot;</span><span class="s1">&gt;</span><span class="s0">pygame module to control the display window and screen</span><span class="s1">&lt;/span&gt;&lt;/a&gt; </span><span class="s0">module is</span>
<span class="s0">a binding around the SDL library, SDL is really doing all this work.</span><span class="s1">&lt;/p&gt;</span>
<span class="s1">&lt;p&gt;</span><span class="s0">There are advantages and disadvantages to setting the display mode in this</span>
<span class="s0">manner.</span>
<span class="s0">The advantage is that if your game requires a specific display mode,</span>
<span class="s0">your game will run on platforms that do not support your requirements.</span>
<span class="s0">It also makes life easier when you're getting something started,</span>
<span class="s0">it is always easy to go back later and make the mode selection a little more</span>
<span class="s0">particular.</span>
<span class="s0">The disadvantage is that what you request is not always what you will get.</span>
<span class="s0">There is also a performance penalty when the display mode must be emulated.</span>
<span class="s0">This tutorial will help you understand the different methods for querying</span>
<span class="s0">the platforms display capabilities, and setting the display mode for your game.</span><span class="s1">&lt;/p&gt;</span>

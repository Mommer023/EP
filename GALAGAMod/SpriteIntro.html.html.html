<span class="s1">&lt;p&gt;</span><span class="s0">At this point the two classes seem pretty basic. Not doing a lot more than you</span>

<span class="s0">the enemies, checking for the </span><span class="s4">&amp;quot;</span><span class="s0">close_to_player</span><span class="s4">&amp;quot; </span><span class="s0">flag. Later on your game could</span>
<span class="s0">add multiple players, and instead of adding more </span><span class="s4">&amp;quot;</span><span class="s0">close_to_player2</span><span class="s4">&amp;quot;</span><span class="s0">,</span>

<span class="s0">each player.</span><span class="s1">&lt;/p&gt;</span>
<span class="s1">&lt;p&gt;</span><span class="s0">Another important benefit of using the </span><span class="s1">&lt;code </span><span class="s2">class</span><span class="s3">=&quot;docutils literal notranslate&quot;</span><span class="s1">&gt;&lt;span </span><span class="s2">class</span><span class="s3">=&quot;pre&quot;</span><span class="s1">&gt;</span><span class="s0">Sprites</span><span class="s1">&lt;/span&gt;&lt;/code&gt; </span><span class="s0">and </span><span class="s1">&lt;code </span><span class="s2">class</span><span class="s3">=&quot;docutils literal notranslate&quot;</span><span class="s1">&gt;&lt;span </span><span class="s2">class</span><span class="s3">=&quot;pre&quot;</span><span class="s1">&gt;</span><span class="s0">Groups</span><span class="s1">&lt;/span&gt;&lt;/code&gt; </span><span class="s0">is that the groups</span>

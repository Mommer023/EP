<span class="s0">Tutorials</span>
<span class="s0">---------</span>

<span class="s0">:doc:`Introduction to Pygame &lt;tut/PygameIntro&gt;`</span>
  <span class="s0">An introduction to the basics of pygame.</span>
  <span class="s0">This is written for users of Python and appeared in volume two of the Py magazine.</span>

<span class="s0">:doc:`Import and Initialize &lt;tut/ImportInit&gt;`</span>
  <span class="s0">The beginning steps on importing and initializing pygame.</span>
  <span class="s0">The pygame package is made of several modules.</span>
  <span class="s0">Some modules are not included on all platforms.</span>

<span class="s0">:doc:`How do I move an Image? &lt;tut/MoveIt&gt;`</span>
  <span class="s0">A basic tutorial that covers the concepts behind 2D computer animation.</span>
  <span class="s0">Information about drawing and clearing objects to make them appear animated.</span>

<span class="s0">:doc:`Chimp Tutorial, Line by Line &lt;tut/ChimpLineByLine&gt;`</span>
  <span class="s0">The pygame examples include a simple program with an interactive fist and a chimpanzee.</span>
  <span class="s0">This was inspired by the annoying flash banner of the early 2000s.</span>
  <span class="s0">This tutorial examines every line of code used in the example.</span>

<span class="s0">:doc:`Sprite Module Introduction &lt;tut/SpriteIntro&gt;`</span>
  <span class="s0">Pygame includes a higher level sprite module to help organize games.</span>
  <span class="s0">The sprite module includes several classes that help manage details found in almost all games types.</span>
  <span class="s0">The Sprite classes are a bit more advanced than the regular pygame modules,</span>
  <span class="s0">and need more understanding to be properly used.</span>

<span class="s0">:doc:`Surfarray Introduction &lt;tut/SurfarrayIntro&gt;`</span>
  <span class="s0">Pygame used the NumPy python module to allow efficient per pixel effects on images.</span>
  <span class="s0">Using the surface arrays is an advanced feature that allows custom effects and filters.</span>
  <span class="s0">This also examines some of the simple effects from the pygame example, arraydemo.py.</span>

<span class="s0">:doc:`Camera Module Introduction &lt;tut/CameraIntro&gt;`</span>
  <span class="s0">Pygame, as of 1.9, has a camera module that allows you to capture images,</span>
  <span class="s0">watch live streams, and do some basic computer vision.</span>
  <span class="s0">This tutorial covers those use cases.</span>

<span class="s0">:doc:`Newbie Guide &lt;tut/newbieguide&gt;`</span>
  <span class="s0">A list of thirteen helpful tips for people to get comfortable using pygame.</span>

<span class="s0">:doc:`Making Games Tutorial &lt;tut/MakeGames&gt;`</span>
  <span class="s0">A large tutorial that covers the bigger topics needed to create an entire game.</span>

<span class="s0">:doc:`Display Modes &lt;tut/DisplayModes&gt;`</span>
  <span class="s0">Getting a display surface for the screen.</span>

<span class="s0">:doc:`한국어 튜토리얼 (Korean Tutorial) &lt;tut/ko/빨간블록 검은블록/개요&gt;`</span>
  <span class="s0">빨간블록 검은블록</span>

<span class="s0">:doc:`Tutorial de Pygame - Ejemplo del Chimpancé, Línea Por Línea &lt;tut/es/ChimpanceLineaporLinea&gt;`</span>
  <span class="s0">Los ejemplos de pygame incluyen un sencillo programa con un puño interactivo y un chimpancé.</span>
  <span class="s0">Está inspirado en el molesto banner de flash de principios de la década de 2000.</span>
  <span class="s0">Este tutorial examina cada línea de código utilizada en el ejemplo.</span>

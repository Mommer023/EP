<span class="s1">&lt;p&gt;</span><span class="s0">The Overlay objects provide support for accessing hardware video overlays.</span>
<span class="s0">Video overlays do not use standard </span><span class="s1">&lt;code </span><span class="s2">class</span><span class="s3">=&quot;docutils literal notranslate&quot;</span><span class="s1">&gt;&lt;span </span><span class="s2">class</span><span class="s3">=&quot;pre&quot;</span><span class="s1">&gt;</span><span class="s0">RGB</span><span class="s1">&lt;/span&gt;&lt;/code&gt; </span><span class="s0">pixel formats, and can use</span>
<span class="s0">multiple resolutions of data to create a single image.</span><span class="s1">&lt;/p&gt;</span>
<span class="s1">&lt;p&gt;</span><span class="s0">The Overlay objects represent lower level access to the display hardware. To</span>
<span class="s0">use the object you must understand the technical details of video overlays.</span><span class="s1">&lt;/p&gt;</span>
<span class="s1">&lt;p&gt;</span><span class="s0">The Overlay format determines the type of pixel data used. Not all hardware</span>
<span class="s0">will support all types of overlay formats. Here is a list of available</span>
<span class="s0">format types:</span><span class="s1">&lt;/p&gt;</span>
<span class="s1">&lt;div </span><span class="s2">class</span><span class="s3">=&quot;highlight-default notranslate&quot;</span><span class="s1">&gt;&lt;div </span><span class="s2">class</span><span class="s3">=&quot;highlight&quot;</span><span class="s1">&gt;&lt;pre&gt;&lt;span&gt;&lt;/span&gt;&lt;span </span><span class="s2">class</span><span class="s3">=&quot;n&quot;</span><span class="s1">&gt;</span><span class="s0">YV12_OVERLAY</span><span class="s1">&lt;/span&gt;&lt;span </span><span class="s2">class</span><span class="s3">=&quot;p&quot;</span><span class="s1">&gt;</span><span class="s0">,</span><span class="s1">&lt;/span&gt; &lt;span </span><span class="s2">class</span><span class="s3">=&quot;n&quot;</span><span class="s1">&gt;</span><span class="s0">IYUV_OVERLAY</span><span class="s1">&lt;/span&gt;&lt;span </span><span class="s2">class</span><span class="s3">=&quot;p&quot;</span><span class="s1">&gt;</span><span class="s0">,</span><span class="s1">&lt;/span&gt; &lt;span </span><span class="s2">class</span><span class="s3">=&quot;n&quot;</span><span class="s1">&gt;</span><span class="s0">YUY2_OVERLAY</span><span class="s1">&lt;/span&gt;&lt;span </span><span class="s2">class</span><span class="s3">=&quot;p&quot;</span><span class="s1">&gt;</span><span class="s0">,</span><span class="s1">&lt;/span&gt; &lt;span </span><span class="s2">class</span><span class="s3">=&quot;n&quot;</span><span class="s1">&gt;</span><span class="s0">UYVY_OVERLAY</span><span class="s1">&lt;/span&gt;&lt;span </span><span class="s2">class</span><span class="s3">=&quot;p&quot;</span><span class="s1">&gt;</span><span class="s0">,</span><span class="s1">&lt;/span&gt; &lt;span </span><span class="s2">class</span><span class="s3">=&quot;n&quot;</span><span class="s1">&gt;</span><span class="s0">YVYU_OVERLAY</span><span class="s1">&lt;/span&gt;</span>

<span class="s1">&lt;p&gt;</span><span class="s0">The width and height arguments control the size for the overlay image data.</span>
<span class="s0">The overlay image can be displayed at any size, not just the resolution of</span>
<span class="s0">the overlay.</span><span class="s1">&lt;/p&gt;</span>
<span class="s1">&lt;p&gt;</span><span class="s0">The overlay objects are always visible, and always show above the regular</span>

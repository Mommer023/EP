<span class="s1">&lt;span </span><span class="s2">id</span><span class="s3">=&quot;id1&quot;</span><span class="s1">&gt;&lt;/span&gt;&lt;h2&gt;</span><span class="s0">5. User-controllable objects</span><span class="s1">&lt;a </span><span class="s2">class</span><span class="s3">=&quot;headerlink&quot; </span><span class="s2">href</span><span class="s3">=&quot;#makegames-5&quot; </span><span class="s2">title</span><span class="s3">=&quot;Permalink to this heading&quot;</span><span class="s1">&gt;</span><span class="s0">¶</span><span class="s1">&lt;/a&gt;&lt;/h2&gt;</span>
<span class="s1">&lt;p&gt;</span><span class="s0">So far you can create a Pygame window, and render a ball that will fly across the screen. The next step is to make some bats which</span>
<span class="s0">the user can control. This is potentially far more simple than the ball, because it requires no physics (unless your user-controlled</span>
<span class="s0">object will move in ways more complex than up and down, e.g. a platform character like Mario, in which case you'll need more physics).</span>
<span class="s0">User-controllable objects are pretty easy to create, thanks to Pygame's event queue system, as you'll see.</span><span class="s1">&lt;/p&gt;</span>
<span class="s1">&lt;section </span><span class="s2">id</span><span class="s3">=&quot;a-simple-bat-class&quot;</span><span class="s1">&gt;</span>
<span class="s1">&lt;span </span><span class="s2">id</span><span class="s3">=&quot;makegames-5-1&quot;</span><span class="s1">&gt;&lt;/span&gt;&lt;h3&gt;</span><span class="s0">5.1. A simple bat class</span><span class="s1">&lt;a </span><span class="s2">class</span><span class="s3">=&quot;headerlink&quot; </span><span class="s2">href</span><span class="s3">=&quot;#a-simple-bat-class&quot; </span><span class="s2">title</span><span class="s3">=&quot;Permalink to this heading&quot;</span><span class="s1">&gt;</span><span class="s0">¶</span><span class="s1">&lt;/a&gt;&lt;/h3&gt;</span>
<span class="s1">&lt;p&gt;</span><span class="s0">The principle behind the bat class is similar to that of the ball class. You need an </span><span class="s1">&lt;code </span><span class="s2">class</span><span class="s3">=&quot;docutils literal notranslate&quot;</span><span class="s1">&gt;&lt;span </span><span class="s2">class</span><span class="s3">=&quot;pre&quot;</span><span class="s1">&gt;</span><span class="s0">__init__</span><span class="s1">&lt;/span&gt;&lt;/code&gt; </span><span class="s0">function to initialise the</span>
